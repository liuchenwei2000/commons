commons-lang 用来处理Java基本对象方法的工具类包，可以简化很多平时经常要用到的写法，例如判断字符串是否为空等等。


commons.lang包共包含了17个实用的类：
 
ArrayUtils C 用于对数组的操作，如添加、查找、删除、子数组、倒序、元素类型转换等；
BitField C 用于操作位元，提供了一些方便而安全的方法；
BooleanUtils C 用于操作和转换boolean或者Boolean及相应的数组；
CharEncoding C 包含了Java环境支持的字符编码，提供是否支持某种编码的判断；
CharRange C 用于设定字符范围并做相应检查；
CharSet C 用于设定一组字符作为范围并做相应检查；
CharSetUtils C 用于操作CharSet；
CharUtils C 用于操作char值和Character对象；
ClassUtils C 用于对Java类的操作，不使用反射；
ObjectUtils C 用于操作Java对象，提供null安全的访问和其他一些功能；
RandomStringUtils C 用于生成随机的字符串；
SerializationUtils C 用于处理对象序列化，提供比一般Java序列化更高级的处理能力；
StringEscapeUtils C 用于正确处理转义字符，产生正确的Java、JavaScript、HTML、XML和SQL代码；
StringUtils C 处理String的核心类，提供了相当多的功能；
SystemUtils C 在java.lang.System基础上提供更方便的访问，如用户路径、Java版本、时区、操作系统等判断；
Validate C 提供验证的操作，有点类似assert断言；
WordUtils C 用于处理单词大小写、换行等。
